你能构造出连续值的最大数目

数学归纳法思想：

  1. 极端构造序列：1,2,4,8,…
  2. 一定无法构造：sum(i) + 1 < a[i+1]，一定会停止
  3. a[1] = 1，从0-1都可构造，否则连续值为0.
  4. a[2] = 1或2，从0-sum(2)都可构造出来
  5. 如果0-sum(i-1)都能构造出来，a[i]不是停止点，那么sum(i-1)+a[i]=sum(i)就能构造出来
    综上，若第一个停止点为a[i]，则最大构造为0-sum(i-1)，答案即sum(i-1)+1
